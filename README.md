# ii-eve Cheatsheet Keybinds

Keybinds page for the [ii-eve](https://github.com/djOB2EOTWQW1/ii-eve) cheatsheet, packaged as an extension via the `cheatsheetPages` contribution point.

## Install
Extensions settings → paste this directory's path (local) or the repo URL → Install → enable. The tab appears in the cheatsheet.
